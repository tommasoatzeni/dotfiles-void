# dotfiles-void
Configurazione minimale per Void Linux su ThinkPad X230 (pataccone).

## Hardware
- ThinkPad X230, i5-3320M, 16GB RAM, 1TB SSD Samsung 870 QVO
- Font TTY: ter-v16b
- Shell: bash

## Contenuto
- `home/` — dotfiles utente (.bashrc, .tmux.conf, .vimrc)
- `etc/rc.conf` — font e keymap TTY
- `etc/dnsmasq.d/lab.conf` — router portatile 4G (su mucchina)

## Installazione
```bash
./install.sh
```
